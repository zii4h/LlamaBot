## ZBot Web-Assistant Chatbot
a lightweight web chatbot that runs locally using Ollama models and can be dropped into any website with minimal setup 
> PS: Includes a demo landing page for showcase 

## If you want to try it:
1. Download and install app from: **https://ollama.com**
2. Run: ```ollama pull llama3.2``` on CMD or Powershell
3. Open **Ollama app** in the background
4. Open ```index.html``` with Live Server

## Model Options (Ollama Runtime)

| Model | Description |
|------|-------------|
| mistral | 7B instruct model optimized for fast token generation and low-latency chat responses |
| llama3.1 | Meta instruction model with improved reasoning accuracy and long-context handling at higher compute cost |
| gemma2 | Google instruction model optimized for balanced inference performance and stable output behavior |
| phi3 | Microsoft model used for low-resource setups or local testing with reduced memory usage |
